# YouTube 클론코딩

### Demo : [youtubeclone-lani.netlify.app](https://youtubeclone-lani.netlify.app/)

### 사용 기술

- HTML, CSS

### 개발 목표

- HTML, CSS(Flex Box, media query) 이해를 바탕으로 반응형 웹 구현

### 개발 기간

- 진행중(2022.05.20~)

### 주요기능

- 추천 동영상을 보여줄 때 media query를 적용하여 반응형으로 구현
- youTube Api 활용하여 데이터 조회

### 개선사항

- API Key 환경변수로 변경
